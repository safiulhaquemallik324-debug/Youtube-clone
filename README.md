# 🎥 YouTube Clone

A modern and responsive **YouTube Clone** built using **HTML, SCSS, and JavaScript**. This project replicates the look and feel of YouTube with a clean user interface, smooth interactions, responsive layouts, and beautiful **glassmorphism effects**.

## 🚀 Features

- 🎨 Modern Glassmorphism UI
- 📱 Fully Responsive Design
- 🔍 YouTube-style Search Bar
- 📂 Responsive Sidebar Navigation
- 🎬 Video Card Layout
- ⚡ Smooth Hover Effects & Animations
- 🖥️ Mobile, Tablet & Desktop Friendly
- 🧩 Built with Clean and Organized Code

## 🛠️ Tech Stack

- HTML5
- SCSS (Sass)
- JavaScript (ES6)

## 🎯 What I Learned

- SCSS Workflow & Styling
- CSS Grid & Flexbox
- JavaScript DOM Manipulation
- Glassmorphism Design Techniques
- UI/UX Design Principles
